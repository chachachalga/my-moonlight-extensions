# Gif Sorter - Moonlight Extension
A simple QoL extension that allows users to sort their gifs into custom folders.

## Future features
- Allow adding custom names to gifs for nice compatibility with `Favorite GIF Search` extension.
- Allow drag-and-drop reordering of folders
- "Selection mode" to more easily add a large amount of gifs to a folder
